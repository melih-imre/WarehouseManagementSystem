# WAREHOUSE DATABASE MANAGEMENT SYSTEM

## DESCRIPTION AND RULES
The  **Warehouse Database Management System (WDMS)**  is a Java-based desktop application designed to efficiently manage the operations of a warehouse. 

The system is capable of managing CRUD operations for various warehouse operations, including:

-   **Products:**  Maintain detailed records of products with associated brands, categories, and pricing.
-   **Locations:**  Organize inventory with aisles, shelves, and specific storage locations, while tracking quantities.
-   **Clients:**  Manage client information, including contact details and addresses.
-   **Transactions:**  Record and manage all inventory transactions, ensuring accurate tracking and reporting.

## GETTING STARTED

### INSTALLATION

 1. Clone the repository
	 [WDMS git](https://github.com/melih-imre/WarehouseManagementSystem.git)
	 
 2. Set up JavaFX
	 Configure your IDE for JavaFX SDK 20.
	 
 3. Run the application
	 Start the application by running Main class.


### PREREQUISITES

 1. Java Development Kit (JDK)
 2. JavaFX SDK
 3. Database connection via dbconnect.sh script
